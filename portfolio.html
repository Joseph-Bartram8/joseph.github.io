Joseph Bartram Portfolio Website

My Goals

I wanted to create a website for multiple reasons:

  • I wanted somewhere to showcase the projects I do and also act as an archive so I can look back and see how I've progressed.
  • I also wanted to learn responsive web design as it wasn't something that was taught to us in school.

My Journey

As I knew my main challenge here would be the responsive web design, I did a little research and found a Udemy course on Flexbox by Peter Sommerhoff which looked really good. it took around 2-3 hours and gave me a really good overview of how to use flexbox to create responsive websites.

After a fair amount of experimenting with CSS using felxbox styles. I started to sketch out what I wanted my website to look like. I'd seen some designs I quite liked which I drew inspiration from and wanted the website to relfect not just the more professional work I'm doing, but also who I am and my character.

The Results

I guess this website will never be finished as I will alwasy be adding to it, for instance, even now I'm thinking it would be good to have links to my certificates from the education page.

Importantly I've learnt so much during this project. A little acessibility, responsive web design, a little java script and I've tested this on multiple desktop and mobile browsers. It's not perfect yet, but definitley good enough to release while I work on some other projects

I have tested this website on the following devices: PC (Chrome and Edge), iPhone 8+, iPhone 11 pro
